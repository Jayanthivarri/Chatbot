# AI Chatbot

A simple terminal-based AI chatbot built using Python and the OpenRouter API. The chatbot maintains session memory during runtime and uses a custom system prompt for generating responses.

## Features

- Terminal-based chatbot
- Session memory (current session only)
- Custom system prompt
- User profile support using JSON
- Powered by OpenRouter API
- Colored terminal output using Colorama
